# 수업 내용 #

## git 명령어 ##

#### git config --global user.name "xxxx" ####
사용자명 등록

#### git config --global user.email "xxxx@naver.com" ####
이메일 등록

#### git init ####
현재 폴더를 메인으로 하여 관리 (초기화)

#### touch 파일명 ####
폴더에 파일생성

#### git add 파일명 ####
git에 생성한 파일추가

#### git commit -m "파일 설명" ####
git add한 내용을 설명하는것

#### git status ####
현재 상태 확인

#### git log ####
커밋한 모든 이력 확인
#### git log --oneline ####
커밋한 모든 이력을 간단히 확인 (제목만)
